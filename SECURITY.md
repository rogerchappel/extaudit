# Security Policy

ExtAudit is a local-first extension manifest auditor. Please report suspected security issues privately through GitHub security advisories for `rogerchappel/extaudit`, or by opening an issue with sensitive details removed if the report is not confidential.

## Supported Versions

The latest release and the current `main` branch receive security fixes.

## Reporting Guidance

Include the ExtAudit version or commit, the command you ran, a minimal sanitized fixture if possible, and the expected versus actual behavior. Do not include private extension source, credentials, tokens, or proprietary manifests in public issues.

## Scope

ExtAudit flags suspicious manifest patterns and risky extension metadata. It is not a full malware scanner, sandbox, or substitute for reviewing extension source code and publisher provenance.
